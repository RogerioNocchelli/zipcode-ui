Zipcode API
=======

## Quick start

1 - Clone the repo:

```bash
git clone git@github.com:RogerioNocchelli/zipcode-ui.git
```

2 - Change to the directory created

```bash
cd zipcode-ui/
```

2 - install dependencies

```bash
npm install
```

3 - Run project

```bash
ng serve
```

4 - Navigate to http://localhost:4200/
